## 成员运算符

```cpp
string s1 = "a string", *p = &s1;
auto n = s1.size(); //运行string对象s1的size成员
n = (*p).size();
n = p->size;
```

## sizeof运算符

```cpp
Sales_data data, *p;
sizeof(Sales_data); // 存储Sales_data类型的对象所占的空间大小
sizeof data; //data的类型的大小
sizeof p; // 指针所占的空间
sizeof *p; //p所指类型的空间大小
sizeof data.revenue; // 成员对应类型的大小
```
